# ciao-git

A set of terminal aliases to give your Git commands that special sauce.

## Installation
Clone the repository and run the install script:
```bash
git clone https://github.com/yourusername/ciao-git.git
cd ciao-git
./install.sh
```

## Uninstallation
To completely remove the tool and clean up your system:
```bash
cd ciao-git
./uninstall.sh
```
